#gulp使用说明

##目录结构

- build 
	* css 
	* js
	* img
- css
- js
- img
- less
- src
- index.html
- gulpfile.js

> [1] build/ 用于存储编译（合并压缩）后的.css, .js, .png, .jpg等文件
> [2] css/ 用于存储 less/ 文件夹下.less文件合并编译完成的.css文件
> [3] js/ 用于存储有src/文件夹下的.js文件合并后的Js文件
> [4] less/ 用于存储.less源文件
> [5] src/ 用于存储.js源文件
> [6] gulpfile.js  gulp配置文件
